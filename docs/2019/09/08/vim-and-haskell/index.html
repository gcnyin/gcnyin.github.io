配置vim来写Haskell

Sep 8 vim Comments

第一步当然是安装vim，用所在系统上的包管理器安装即可。

第二步是安装vim haskell插件，但在安装前需要先安装插件管理器。

这里使用vim-plug来安装，按照主页给出的命令安装即可。

安装成功后，.vimrc应该大约是这个样子。

此时就可以添加stylish-haskell插件了。

然后在vim中执行:PlugInstall进行安装。

当然此时这个插件还不能用，因为它依赖命令行工具stylish-haskell。

使用stack安装。

好了，到这里你应该可以愉快地作用vim写Haskell了。
