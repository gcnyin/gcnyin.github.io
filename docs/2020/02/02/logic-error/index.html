一份简单的逻辑清单

Posted on 2020-02-02 In logic

一、前言

最近武汉红会丑闻、中成药治新冠肺炎等事情发生，参与讨论的人非常多，但在围观过程中，发现很多人，包括我，都犯了一些逻辑错误。因此整理成此文，以便日后随时核查，理清自己的逻辑。

二、逻辑错误

（一）稻草人谬误

指B发表了若干言论，A进行了反驳，但在反驳的过程中，歪曲B的言论，把B的论点论据都偷偷换掉了。

“我们要好好学习《英语》。”

“这个人坏透了，他竟然让我们不要去学习《数学》？！”

（二）错误归因

两件事表面存在某种相关性，但并不意味着一件事就是另一件事的原因。

（三）诉诸感情

不是通过有力的论据，而是通过感情，来说服别人。

“狗是人类的好朋友，为什么要吃狗肉？”

（四）谬误谬误

论据是错的，不一定就是说论点是错的。

（五）滑坡谬误

某人用，A发生了，就会发生B，B是不好的、负面的，来论证A是不好的、不应该发生的。

（六）人身攻击

针对对方的品格、地位、动机、处境等等他/她本人的状况进行攻击，来达到否定对方言论的目的。

“这人以前偷过东西，所以我们不应该相信他针对某某领导的举报。”

（七）诉诸虚伪

不去正面回应他人的问题，而是指责他人之前的言论里有某种错误。

“这里是禁烟区，请你不要再在这里抽烟了。”

“你以前不也在这里抽过吗？”

（八）举证责任

A举出抛出一个论点，但自己不去举证，反而让质疑的人去举出反例。

“您在公开场合宣称双黄莲可以治疗新型肺炎，请问有什么依据吗？”

“你有本事先给我举个反例，证明它不生效。”

（九）诉诸权威

A抛出一个论点，当B质疑时，宣称某某权威认同这个观点，所以这个观点是正确的。

“您在公开场合宣称双黄莲可以治疗新型肺炎，请问有什么依据吗？”

“某某研究所和某院士都说了，你还不信？”

（十）没有真正的苏格兰人/诉诸纯洁

A抛出一个全称命题，即所有的XXX都如何如何。当B举出一个反例时，A会宣称这个反倒不是真正的XXX。

“所有的某机构人人员都是好的”

“我怎么看新闻上说，该机构里存在XXX行为”

“这些人都不是真正的XXX人。真正的XXX人是不会做这样事的。”

先总结十条。
